## Features

- Professional bowtie diagram editor
- Risk assessment modeling for HSE
- Barrier-based safety analysis
- Export to PNG and PDF
- Collaborative editing and sharing
- Dark/Light theme support
- Real-time auto-save

## Tech Stack

- React + TypeScript
- React Flow (@xyflow/react)
- Tailwind CSS
- Cloudflare Workers

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Deployment

```bash
# Deploy to Cloudflare Pages
npm run deploy

# Deploy to Netlify
npm run deploy:netlify

# Deploy to Vercel
npm run deploy:vercel
```
